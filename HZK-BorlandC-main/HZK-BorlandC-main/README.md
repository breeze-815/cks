## 说明
`code`文件夹中是在BorlandC+DosBox环境下的调用接口，包括头文件和源文件  
`font`文件夹中是汉字库文件  
`示例工程`文件夹中的示例运行`Borland C++ 3.1.vbs`即可  

#### 关于文件命名
|缩写|说明|
|:-:|:-:|
|ASC|ASCII码库|
|HZK|汉字库|
|S|宋体|
|F|仿宋|
|H|黑体|
|K|楷体|
|\_B|Bold 粗体|
|\_T|Traditional 繁体|

#### 关于24px字库
一般的24px字库是反字字模(早期24px字被常用于印刷，故作成反字字模)，并且文字与标点分两个文件存放。  
这里提供的24px字库已经经过处理，正常使用即可。
